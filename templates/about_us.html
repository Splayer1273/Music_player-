About This Project

This Music Player web application allows users to upload, manage, and listen to their favorite songs in a visually appealing and responsive interface. It is designed with simplicity, performance, and modern design in mind.

Project Features

  • User Registration and Login
  • Upload songs with metadata (title, artist, genre)
  • Audio file storage and playback
  • Responsive UI with a sleek music-themed design
  • Message feedback for user actions (e.g., song added, errors)

Tech Stack

  • Backend: Django (Python)
  • Frontend: HTML, CSS (with Tailwind), JavaScript, Django Templates (Jinja-like)
  • Database: SQLite (can be upgraded to PostgreSQL)
